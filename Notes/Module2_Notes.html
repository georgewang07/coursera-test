  • the StyleSheet can be inclueded into head element

Lecture13: Element, Class, ID, and grouping Selectors

  • Element Selector: just put Element before opening bracket;
  • Class Selector: place . before Class Name
  • ID Selector: place # before ID Name
  • grouping Selector: just put , bewtween each one

Lecture14: Combining Selectors

  • Element with Class: place . between Element Selector and Class Selector to indicate the declaration should be applied to Class of Element
  • Child Selector: place > between Element Selectors to indicate the later one that is the direct child of the former
  • Child's Child Selector: Place 2 >s between 3 elements, the declaration will applied to the 3rd one which is a direct child of the 2nd, which is a direct child of the 1st
  • Descendant Selectors: just place a space between Element Selectors to indicate the later one that is the any-level child of the former
