The commands in Git Bash:

Git Clone means you can clone everything in Github into the local repository

You can use "cd" to open any folder you are indicating

You can use "dir" to see all files in the current open repository

Nomal Process for push the content to the Remote/Github:

  • git add . meaning put the content of the current repository to a place where it will be commited
  • git commit -m "xxx" meaning commit on the added content with the message "xxx" which will come up in Github
  • git push meaning push the committed content to the Remote

    • Notes: if there any issues about connection or proxy, try 2 methods then git push

  • git config --global user.email "wanghao07@outlook.com" git config --global user.name "georgewang07"
  • Or switch the Proxy: git config --global http.proxy http://127.0.0.1:7890 git config --global https.proxy http://127.0.0.1:7890
